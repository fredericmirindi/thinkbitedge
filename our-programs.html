AI Explorers

Intermediate Ages 12-18

AI Explorers is our flagship program designed to introduce students to the fascinating world of artificial intelligence. Through hands-on projects and real-world applications, students will learn the fundamentals of AI, machine learning, and data science in an engaging and accessible way.

Program Highlights

AI Fundamentals

Learn core concepts of artificial intelligence, machine learning algorithms, and neural networks.

Coding with Python

Develop programming skills using Python, the most popular language for AI and data science.

Data Analysis

Learn to collect, analyze, and visualize data to extract meaningful insights.

Practical Projects

Build AI-powered applications like chatbots, image recognition systems, and predictive models.

Who Should Join?

This program is perfect for students who:

  • Are curious about artificial intelligence and its applications
  • Have basic computer skills and an interest in technology
  • Want to develop problem-solving and critical thinking abilities
  • Are interested in pursuing careers in tech, data science, or AI

Program Details

Duration:

10 weeks (20 sessions)

Format:

In-person and virtual options

Class Size:

12-15 students

Prerequisites:

Basic computer skills

Program Curriculum

Our curriculum is designed to build knowledge progressively, starting with fundamentals and advancing to more complex concepts and applications.

Module 1: Introduction to AI (Weeks 1-2)

  • History and evolution of artificial intelligence
  • Types of AI: narrow vs. general AI
  • AI ethics and responsible innovation
  • Introduction to Python programming

Project: Create a simple rule-based AI system

Module 2: Machine Learning Basics (Weeks 3-4)

  • Supervised vs. unsupervised learning
  • Data collection and preparation
  • Training and testing models
  • Introduction to popular ML libraries

Project: Build a simple classification model

Module 3: Neural Networks (Weeks 5-6)

  • Structure and function of neural networks
  • Training neural networks
  • Deep learning concepts
  • Computer vision applications

Project: Create an image recognition application

Module 4: Natural Language Processing (Weeks 7-8)

  • Text processing and analysis
  • Sentiment analysis
  • Chatbot development
  • Language generation

Project: Build a simple conversational AI

Module 5: Final Project (Weeks 9-10)

  • Project planning and design
  • Implementation and testing
  • Presentation skills
  • Future learning pathways

Project: Develop an AI solution for a real-world problem

Learning Outcomes

By the end of this program, students will be able to:

  • Understand core AI and machine learning concepts
  • Write Python code for data analysis and AI applications
  • Build and train simple machine learning models
  • Develop AI-powered applications that solve real problems
  • Critically evaluate AI technologies and their ethical implications

Code Creators

Beginner Ages 10-16

Code Creators is our introductory programming program designed to make coding fun and accessible for beginners. Students learn to build creative projects while developing problem-solving skills and computational thinking.

Program Highlights

Coding Fundamentals

Learn core programming concepts like variables, loops, conditionals, and functions.

Interactive Projects

Create games, animations, and interactive stories that you can share with friends and family.

Collaborative Learning

Work with peers on team projects that solve real community problems.

Creative Problem-Solving

Develop computational thinking and creative approaches to solving challenges.

Who Should Join?

This program is perfect for students who:

  • Are curious about how technology works
  • Enjoy creative activities and building things
  • Want to learn a valuable skill in a fun, supportive environment
  • Have little to no prior coding experience

Program Details

Duration:

8 weeks (16 sessions)

Format:

In-person and virtual options

Class Size:

10-12 students

Prerequisites:

None - beginners welcome!

Program Curriculum

Our curriculum uses block-based and text-based programming to introduce coding concepts in an engaging, visual way.

Module 1: Introduction to Coding (Weeks 1-2)

  • What is coding and how does it work?
  • Block-based programming with Scratch
  • Creating simple animations and stories
  • Basic programming concepts: sequences and events

Project: Interactive digital story

Module 2: Building with Blocks (Weeks 3-4)

  • Loops and repetition
  • Conditionals (if-then statements)
  • Variables and keeping score
  • Game design basics

Project: Simple interactive game

Module 3: Introduction to Text-Based Coding (Weeks 5-6)

  • Transitioning from blocks to text
  • Basic Python syntax
  • Creating interactive programs
  • Functions and reusable code

Project: Interactive quiz or calculator

Module 4: Community Impact Project (Weeks 7-8)

  • Identifying community problems
  • Designing technology solutions
  • Collaborative coding
  • Presenting and sharing your work

Project: Team-based community solution

Learning Outcomes

By the end of this program, students will be able to:

  • Understand fundamental programming concepts
  • Create interactive games and animations
  • Write basic code in a text-based language
  • Apply computational thinking to solve problems
  • Collaborate effectively on coding projects
